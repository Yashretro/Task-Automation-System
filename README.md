--Stock Monitor 

Tracks 10 stocks in real-time. Updates every 10 seconds. Alerts when stocks move big.

-- Stocks I Watch
NVDA, AAPL, MSFT, GOOGL, AMZN, TSLA, META, AMD, INTC, NFLX

--How to Run
```
pip install yfinance
python stock_monitor.py
```

Press Ctrl+C to stop.

--What You Get
- Real-time prices 
- Big move alerts  
- Auto-refresh every 10 seconds
- Clean display
